Recalling: Java

Java is one of if not the hardest languages to learn about compaired to other languages like python or c++.

Part of this stems from Java being an overly verbose language that requires alot of redundency and repetiveness to complete simple code.

  • Java is an a Object Oriented Program. Enabling use for many dot operatons and direct minipulation
  • It's STATICLY typed. Meaning you need to define every single variable as an Int, String, Double, ect.
  • Java supports type casting as long as the types are compatable
  • This language supports overloading constructors and methods with various inputs
  • Java uses curly braces to nest code blocks and seperates lines with the ";" key
  • updates to java may break pre-existing code
