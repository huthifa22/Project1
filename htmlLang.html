Recalling: HTML

Html on the surface seems to be an easy markup language until you learn that its basicly 4 languages wraped into one.

Html alone won't get you anywhere unless you combine it together with CSS, JavaScript, and Php

Html Basics:

  • The base shell of an Html website is comprised of a <!DOCTYPE> heading followed with a matching pair of <head>, and <body> tags
  • Almost everything built in Html come in bracket pairs. Link and inline style are an example of the few that don't.
  • In plain Html the bulk of the work is creating containers of data that contain information with the occational input method.
  • The primary tags used for word formating is <p>, <h1-3>, <table>, and <li>. With <div> for grouping them all together if needed.
  • In order to give your webpage a bit of appeal. You need to add some inline CSS, or inplement a CSS style Sheet.

Css Basics:

JavaScript not to be confused with Java, is the brain of the webpage. It puts logic into your inputs and text.

  • It is an OOP language relying on functions and methods
  • It has unique event listeners to capture the users interactions with the page
  • One of the downsides of this language is that you can't have overloader constructors

Php is roughly an extension to js as its a traditional programming language with a few quirks

Php Basics:

  • One of Php's defining features is the echo command. It allows for direct DOM modification
  • All declared variables start with the $ symbol and is a dynamic type.
  • It is the preferred method of storing arrays of data
  • Php's second niche is the ability to connect your website to an exteral server and transfer data
  • Strings are concattinated with the . operator
