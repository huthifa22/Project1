Recalling: HTML

Html on the surface seems to be an easy markup language until you learn that its basicly 4 languages wraped into one.

Html alone won't get you anywhere unless you combine it together with CSS, JavaScript, and Php

Html Basics:

  • The base shell of an Html website is comprised of a <!DOCTYPE> heading followed with a matching pair of <head>, and <body> tags
  • Almost everything built in Html come in bracket pairs. Link and inline style are an example of the few that don't.
  • In plain Html the bulk of the work is creating containers of data that contain information with the occational input method.
  • The primary tags used for word formating is <p>, <h1-3>, <table>, and <li>. With <div> for grouping them all together if needed.
  • In order to give your webpage a bit of appeal. You need to add some inline CSS, or inplement a CSS style Sheet.
